Проверка перекрытия слайдов видео-окном

Инструмент анализирует .pptx и подсвечивает текстовые блоки, попадающие в область видео спикера (picture-in-picture).

PHP 8 + PhpPresentation

Загрузка презентации

GitHub Pages не поддерживает загрузку файлов на сервер. Чтобы выполнить анализ, запустите локально: composer install и php -S 0.0.0.0:8000 -t public.

Что делает анализатор

  • Вычисляет размеры каждого слайда.
  • Извлекает координаты и размеры текстовых блоков.
  • Проверяет пересечения с зоной видео-окна.
  • Готовит HTML-отчёт и JSON для API.
Исходники доступны в репозитории PowerPointCheckSlidesforVideo.
