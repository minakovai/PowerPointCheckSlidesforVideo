# PowerPointCheckSlidesforVideo

## Быстрый старт

1. Установите зависимости:
   ```bash
   composer install
   ```
2. Запустите локальный сервер:
   ```bash
   php -S 0.0.0.0:8000 -t public
   ```
3. Откройте `http://127.0.0.1:8000` и загрузите `.pptx`.

## GitHub Pages

Страница `https://minakovai.github.io/PowerPointCheckSlidesforVideo/` показывает статичное описание и кнопку загрузки,
но фактический анализ доступен только при запуске локального PHP-сервера (GitHub Pages не выполняет PHP).
